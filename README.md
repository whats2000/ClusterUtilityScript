# Cluster Utility Scripts

A collection of useful bash scripts to enhance your cluster experience.

## Available Scripts

### GPU Monitor - nvnodetop

A utility script that monitors the GPU usage of all nodes in a cluster and displays the information in a table. Check the [script](Bash/GPU/nvnodetop/scripts/nvnodetop.sh) for more information.

![nvnodetop](Bash/GPU/nvnodetop/images/nvnodetop.png)

### File Transfer - verisync

A utility script for interactive directory transfer with checksum verification over SSH. Check the [script](Bash/File/verisync/verisync.sh) for more information.

## Getting Started

To use these scripts:

1. Navigate to the directory where you want to use the script
2. Copy and paste the script you need
3. Press Enter to execute
